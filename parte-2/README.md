# Landing Page — Curso de Python com IA

Landing page desenvolvida em Next.js para apresentar um curso de Python com foco em prática, projetos reais e IA.

[https://asimov-test-parte-2.vercel.app](https://asimov-test-parte-2.vercel.app/)

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Bun

## Rodando localmente

```bash
bun dev
```

A aplicação ficará disponível em `http://localhost:3000`.

## Créditos de produção

- **Referência de design:** Google Stitch
- **Codificação:** OpenClaude com Codex (`gpt-5.4 high`)
- **Geração de vídeo:** Google VEO 3

## Observações

- A hero possui vídeo controlado por scroll.
- A section abaixo apresenta o conteúdo que o aluno vai aprender.
- O botão da hero faz scroll suave até a section de conteúdo.
